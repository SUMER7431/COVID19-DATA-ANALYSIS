COVID-19 Data Analysis---

Project Overview--
Global COVID-19 data analysis covering 200+ countries using Python and real world data from Our World in Data.

Tools Used--
 Python
 Pandas
 Matplotlib
 Seaborn
 Jupyter Notebook
 
Dataset--
Our World in Data COVID-19 Dataset
 44,785 rows
 67 columns
 Real updated global data

Key Insights--
 United States had highest total COVID cases globally
 India showed massive case spike in 2021 second wave
 Total deaths strongly correlated with total cases
 Daily new deaths peaked differently across countries
 Brazil and Russia also among top affected countries
